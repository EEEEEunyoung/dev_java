원시형 변수
char, byte, short, int, long, float, double - 정수형, 실수형 = 크기가 다르다. 표현할 수 있는 범위가 다르다. scope가 다르다. 
boolean - 제어문에서 중요함 - 흐름을 다르게 할 때 - 분기문(if, switch)
호출하면 값이 나온다
원시형 변수에는 기본값이 있다. 
원시형 변수를 초기화 할 때는 대입연산자를 사용한다.
변수의 사용목적은 일괄처리에 있다.

원시형 변수는 한번에 하나만 담을 수 있다.
[원시형  타입 정리]
**변수와 메소드를 가질 수 없음
**참조형이 아니므로  변수와 메소드를 소유(=선언=간섭) 할 수 없다.
**원시형 타입은 호출하면 값이 출력이 되니까, 변수와 메소드를 선언할 수 없다.
**원시형 변수는 직접참조방식이므로 값이 호출된다.

[첨조형타입]
