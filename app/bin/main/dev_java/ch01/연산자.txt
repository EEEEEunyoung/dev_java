 if 문  이 하나일 경우, 좌중괄호 우중괄호 생략가능


 &, && 공통점
 - 두 조건 모두 참일때만, 실행문이 실행됨.
  차이점
  - 한 개이면, 앞에 조건이 거짓이더라도 뒤에 조건을 따진다. 
  - 두 개이면, 앞에 조건이 거짓인 경우 판정이 가능하므로 
    굳이 뒤에 조건을 따지지 않는다. - 일량이 줄어든다. 
     T T - TRUE
     F T - FALSE
     T F - FALSE
     F F - FALSE


     ||  - 조건 둘중 한쪽만 참이면, 참
